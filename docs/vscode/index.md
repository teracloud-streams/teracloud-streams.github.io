# Teracloud Streams VS Code Extension

Welcome to the Teracloud Streams VS Code Extension documentation!

## Overview

The Teracloud Streams VS Code Extension provides several helpful tools for SPL language support, application workflows, instance management, and more to assist Streams application developers. See the full overview of the extension and its features on the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=teracloud-aps.teracloud-streams).

This documentation provides supplemental information about features, guides, tips, and additional references to complement the extension's built-in help and walkthroughs.

## Getting started

=== "What's a Streams app?"
    If you're not ready to delve into development yet, check out [Developing Stream Applications](../gettingstarted/index.md) to get familiar with Stream application concepts.
