# Streams Developer Tools Documentation and Blog

This repository contains supplementary developer documentation and a blog built using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

See the published documentation [here](https://teracloud-streams.github.io/).

## Building the doc

A Python 3 environment is required to build the documentation. Use the following command to install the necessary libraries:

```bash
pip install -r requirements.txt
```

### Commands

Contributers can use the following commands to build and host the website locally:

* `mkdocs build` - Build the documentation site.
* `mkdocs serve` - Start the live-reloading docs server.
